# 🧮 Age Calculator App

A simple and responsive **Age Calculator** built using **React** and **Tailwind CSS**.  
This project was developed as a challenge from **Frontend Mentor** to improve frontend development skills and UI implementation.

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/your-username/age-calculator.git

2. Navigate to project folder:
   cd age-calculator

3. Install dependencies:
   npm install

4. Run the app:
   npm run dev
